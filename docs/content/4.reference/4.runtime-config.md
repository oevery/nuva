---

Nuva 最终把可序列化配置合并到 `runtimeConfig.public.nuva`。

## 结构

```ts
interface NuvaPublicConfig {
  api: NuvaApiConfig
  auth: NuvaAuthConfig
}
```

## 客户端读取

```ts
const nuva = useNuvaConfig()
```

## 服务端读取

```ts
import { getNuvaConfig } from '@oevery/nuva/server/utils/config'

export default defineEventHandler((event) => {
  const nuva = getNuvaConfig(event)
  return nuva.api.baseURL
})
```

## 远程请求解析

服务端 `getNuvaConfig(event)` 会把这些字段从字符串解析成结构化请求对象：

- `auth.user.remote.request`
- `auth.permission.remote.request`
- `auth.accessMenu.remote.request`
