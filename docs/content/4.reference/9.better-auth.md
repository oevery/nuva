Better Auth 是 Nuva 的可选 adapter module。

## 模块入口

```ts
export default defineNuxtConfig({
  modules: ['@oevery/nuva/better-auth'],
})
```

模块行为：

- 自动安装 Auth core。
- 自动注入 `useBetterAuthClient()`。
- 注册 Nitro server handler：`basePath/**`。
- 注册 provider 为 `better-auth` 的 server adapter。

## `useBetterAuthClient()`

```ts
const authClient = useBetterAuthClient()
const { data: session } = authClient.useSession()
```

特点：

- SSR 中创建 request-scoped client。
- 自动转发 `cookie` header。
- 客户端按 baseURL、basePath 和 organization 配置缓存 client。
- 根据 `betterAuth.organization` 配置挂载 `organizationClient()`。

## `registerBetterAuthServerAdapter()`

```ts
import { registerBetterAuthServerAdapter } from '@oevery/nuva/better-auth/server-adapter'

registerBetterAuthServerAdapter({
  auth,
  basePath: '/api/auth',
})
```

模块启用时通常会自动注册，业务项目一般不需要手动调用。

## 配置字段

见 [Nuva Config](/reference/nuva-config) 的 `auth.betterAuth`。
