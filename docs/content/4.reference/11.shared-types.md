---
title: Shared Types
description: shared 目录中推荐维护的通用类型和 schema。
---

Nuva 不强制规定业务类型名称，但推荐在 template 或业务项目中维护这些基础结构。

## 分页参数

```ts
export interface PageParams {
  pageNum: number
  pageSize: number
}
```

## 分页结果

```ts
export interface PageResult<T> {
  list: T[]
  total: number
  pageNum: number
  pageSize: number
}
```

## 统一响应

```ts
export interface ApiResponse<T = unknown> {
  code: number | string
  message: string
  data: T
}
```

## 业务 schema

```ts
import * as v from 'valibot'

export const userFormSchema = v.object({
  name: v.pipe(v.string(), v.minLength(2, '名称至少 2 个字符')),
})
```
