---
title: Nuxt Config
description: 在 Nuxt 项目中继承 Nuva layer 并按需启用模块。
---

## 继承 Nuva layer

```ts
export default defineNuxtConfig({
  extends: ['@oevery/nuva'],
})
```

## 启用 Auth core

```ts
export default defineNuxtConfig({
  extends: ['@oevery/nuva'],
  modules: ['@oevery/nuva/auth'],
})
```

## 启用 Better Auth adapter

```ts
export default defineNuxtConfig({
  extends: ['@oevery/nuva'],
  modules: ['@oevery/nuva/better-auth'],
})
```

## 文档站示例

当前文档站使用：

```ts
export default defineNuxtConfig({
  extends: ['@oevery/nuva'],
  srcDir: 'app',
  modules: [
    '@nuxt/ui',
    '@barzhsieh/nuxt-content-mermaid',
    '@nuxt/content',
  ],
  ui: {
    content: true,
  },
})
```
