---
title: 命令速查
description: Nuva monorepo 维护时常用 pnpm 脚本。
---

这些命令面向 Nuva 仓库维护者。业务项目使用者以自己项目的 `package.json` 为准。

## 根目录命令

| 命令 | 说明 |
| --- | --- |
| `pnpm check` | 执行 lint 和 typecheck。 |
| `pnpm lint` | ESLint 检查整个仓库。 |
| `pnpm lint:fix` | 自动修复可修复的 lint 问题。 |
| `pnpm typecheck` | 对所有 workspace 执行 typecheck。 |

## Core 命令

| 命令 | 说明 |
| --- | --- |
| `pnpm core:prepare` | 运行 core 的 `nuxt prepare`。 |
| `pnpm core:typecheck` | core 类型检查。 |
| `pnpm core:test` | core 单测和 e2e。 |
| `pnpm core:build` | core 检查。 |

## Template 命令

| 命令 | 说明 |
| --- | --- |
| `pnpm template:dev` | 启动 template 开发服务。 |
| `pnpm template:build` | 构建 template。 |
| `pnpm template:generate` | 静态生成 template。 |
| `pnpm template:typecheck` | template 类型检查。 |

## Docs 命令

| 命令 | 说明 |
| --- | --- |
| `pnpm docs:dev` | 启动文档站开发服务。 |
| `pnpm docs:build` | 构建文档站。 |
| `pnpm docs:generate` | 静态生成文档站。 |
| `pnpm docs:typecheck` | 文档站类型检查。 |
