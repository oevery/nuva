---
title: 测试与检查
description: 维护 Nuva 仓库时应运行的 lint、typecheck、unit、e2e 和构建命令。
---

## 根目录检查

```bash
pnpm lint
pnpm typecheck
pnpm check
```

## Core 检查

```bash
pnpm core:prepare
pnpm core:typecheck
pnpm core:test
pnpm core:build
```

core 包包含 runtime 和 e2e 检查。修改 core 后应优先跑 core 相关命令。

## Template 检查

```bash
pnpm template:typecheck
pnpm template:build
pnpm template:generate
```

## Docs 检查

```bash
pnpm docs:typecheck
pnpm docs:build
pnpm docs:generate
```

涉及文档信息架构、frontmatter 或 `nuxt-llms` 配置时，还应本地检查：

```txt
/llms.txt
/raw/getting-started/quick-start.md
```

## 推荐策略

- 修改 core：跑 `core:typecheck` 和 `core:test`。
- 修改 template：跑 `template:typecheck` 和 `template:build`。
- 修改 docs：跑 `docs:typecheck` 和 `docs:build`。
- 修改 workspace 依赖或配置：跑根目录 `pnpm check`。
