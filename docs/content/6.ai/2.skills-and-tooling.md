模板项目可以内置 skills 管理，让 AI 工具更稳定地理解项目技术栈。

## 安装 skills

```bash
pnpx skills experimental_install
```

更新 skills：

```bash
pnpx skills update
```

## 模板项目使用

通过 giget 下载模板后：

```bash
pnpm install
pnpm skills:install
```

## 提交约定

- 提交 `skills.json` 和 `skills-lock.json`。
- 不提交 `.agents/`，它是本地安装产物。
- 如需离线模板或强冻结 skills，再考虑提交 `.agents/`。

## 协作建议

- AI 生成代码后，优先让它落到 `app`、`server` 或 `shared` 的真实结构里。
- 不要把临时探索结论长期停留在对话里。
- 新增约定影响业务协作时，优先更新项目内文档。
