title: 新增分页列表
description: 使用 shared 分页类型、server API 和 alova usePagination 完成分页列表。
---

## 目标

从 `shared`、`server/api`、`app/composables/apis` 到页面完成一个分页列表。

## 文件位置

```txt
shared/api/types.ts
shared/api/user.ts
server/api/users.get.ts
app/composables/apis/user.ts
app/pages/users.vue
```

## 步骤

### 1. 定义分页协议

```ts
// shared/api/types.ts
export interface PageParams {
  pageNum: number
  pageSize: number
}

export interface PageResult<T> {
  list: T[]
  total: number
  pageNum: number
  pageSize: number
}
```

### 2. 实现服务端接口

```ts
// server/api/users.get.ts
import type { PageResult } from '#shared/api/types'
import type { User } from '#shared/api/user'
import { ok } from '../utils/response'

export default defineEventHandler((event) => {
  const query = getQuery(event)
  const pageNum = Number(query.pageNum || 1)
  const pageSize = Number(query.pageSize || 10)

  return ok<PageResult<User>>({
    list: [],
    total: 0,
    pageNum,
    pageSize,
  })
})
```

### 3. 封装列表状态

```ts
// app/composables/apis/user.ts
import type { PageParams, PageResult } from '#shared/api/types'
import type { User } from '#shared/api/user'
import { usePagination } from 'alova/client'

export function useUserApi() {
  const http = useHttpClient()

  return {
    list: (params: PageParams) => http.Get<PageResult<User>>('/users', { params }),
  }
}

export function useUserTable() {
  const userApi = useUserApi()

  return usePagination(
    (page, pageSize) => userApi.list({ pageNum: page, pageSize }),
    {
      immediate: true,
      data: response => response.list,
      total: response => response.total,
    },
  )
}
```

### 4. 页面消费

```vue
<script setup lang="ts">
const { data, loading, page, pageSize, total } = useUserTable()
</script>
```

## 常见问题

- 页面层不要直接拼分页接口。
- 分页协议建议放在 `shared`，避免前后端字段漂移。
- 如果后端分页字段不同，优先在 API composable 中适配，不要散落到页面。
