## 配方格式

每篇配方都尽量保持同一结构：

1. 目标。
2. 前置条件。
3. 文件位置。
4. 步骤。
5. 常见问题。
6. 下一步。

如果你想理解为什么这么做，先看 [指南](/guide)。如果你只想查 API，直接看 [参考](/reference)。
