---

Nuva 已接入 VueUse。常用组合式函数可以在业务代码中直接使用 Nuxt auto-import。

## 示例

```ts
const { width } = useWindowSize()
const keyword = useLocalStorage('keyword', '')
const debouncedSearch = useDebounceFn(search, 300)
```

## 适合场景

- 本地存储：`useLocalStorage`。
- 窗口尺寸：`useWindowSize`。
- 防抖节流：`useDebounceFn`、`useThrottleFn`。
- DOM 交互：`useEventListener`、`useElementSize`。

## 什么时候需要额外安装

大多数情况下不需要额外安装。如果你的项目需要显式从 `@vueuse/core` 命名导入，再在项目里补依赖。
