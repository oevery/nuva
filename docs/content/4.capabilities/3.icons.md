---
title: 图标
description: 使用 Nuxt Icon 在业务组件里渲染图标。
---

Nuva 已接入 Nuxt Icon。业务组件可以直接使用 `Icon` 组件。

## 基础用法

```vue
<template>
  <Icon name="lucide:search" />
</template>
```

## 使用建议

- 工具按钮优先使用熟悉的图标，例如搜索、保存、删除、刷新。
- 图标命名遵循 `collection:name`，例如 `lucide:search`。
- 如果项目有固定图标集合，可以在项目配置中补充对应 icon collection。
