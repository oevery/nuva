---
title: 构建与生成
description: 使用 Nuxt build、generate 和 preview 验证你的 Nuva 业务项目。
---

Nuva 项目仍然使用 Nuxt 的构建模型。这里的命令默认在你的业务项目根目录执行。

## 开发服务

```bash
pnpm dev
```

## 生产构建

```bash
pnpm build
```

## 静态生成

```bash
pnpm generate
```

适合静态部署的页面可以使用 generate。涉及 Nuxt server API、动态认证、cookie/session 或实时权限的项目，通常更适合 SSR 部署。

## 预览构建结果

```bash
pnpm preview
```

## 构建前检查

建议至少执行你的项目脚本：

```bash
pnpm lint
pnpm typecheck
pnpm build
```

如果你的项目没有 lint 或 typecheck 脚本，以实际 `package.json` 为准。
