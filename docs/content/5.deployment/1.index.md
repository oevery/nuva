## 上线前最小检查

1. 设置生产环境 API 地址、认证 secret 和 trusted origins。
2. 替换模板 demo auth，不要把 demo auth 当生产认证方案。
3. 确认 `runtimeConfig.public.nuva` 在生产环境值正确。
4. 确认服务端接口有真实鉴权和权限保护。
5. 执行项目自己的 typecheck、lint 和 build。
